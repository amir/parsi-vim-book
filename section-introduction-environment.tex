\section{محیط Vim}
رابط کاربری Vim فاقد هرگونه فهرست انتخاب\Footnote{Menu} یا شمایل\Footnote{Icon} است، در واقع Vim یک رابط کاربری مبتنی بر متن دارد\footnote{gVim نسخه‌ای مبتنی بر GTK+ از Vim است که رابط کاربری گرافیکی را فراهم می‌کند} و با استفاده از فرمانها کنترل می‌شود. به همین دلیل یادگیری Vim در ابتدا مشکل به نظر می‌رسد و اکثر کاربران ترجیح می‌دهند که از ویرایشگرهای دیگر استفاده کنند. اما همینکه مبانی و اصول استفاده از Vim را یاد بگیرید و درک درستی از مفاهیم پا‌یه‌ایی Vim به دست آوردید، روند پیشرفت شما سرعتی دوچندان خواهد گرفت.
